📷 Computer Vision Camera Application
Многофункциональное приложение компьютерного зрения для анализа видеопотока с веб-камеры в реальном времени.

🎮 Режимы работы

    Режим	                                          Описание	                                            Управление
    Simple Camera	                    Базовый захват видео с отображением FPS	                            ESC - выход
    Motion Detection                	Детекция движения с визуализацией	                                ESC - выход
    Color Tracking                    	Отслеживание объектов по цвету	                                G/R/B - выбор цвета
    Color Tracking                      (Multi-Object)	Отслеживание до 10 объектов одновременно	    G/R/B - выбор цвета
    Edge Detection	                    Обнаружение границ алгоритмом Canny	                          ↑/↓ - регулировка порогов
    Face Detection	                    Распознавание лиц и глаз (каскады Хаара)	                        ESC - выход
    Face Detection (DNN)	            Распознавание лиц через нейросеть	                                ESC - выход
    Motion Detection (Threaded)        	Многопоточная детекция движения	                                    ESC - выход
    Record with Effects	                Запись видео с эффектами (сепия, blur, негатив)	         0-4 - эффекты, R - запись, S - скриншот
    Motion Recording                    Автоматическая запись при обнаружении движения	           M - ручная запись, S - остановка
    Body Skeleton	                    Трекинг скелета человека (OpenPose)	                        R - сброс, T - порог уверенности
    ESC -                                     выход из любого режима

✅ Плюсы

    Архитектура
        Модульная структура с разделением на .h и .cpp
        Легко добавлять новые режимы
        Единый паттерн: захват → обработка → вывод
    
    Производительность
        Уменьшенное разрешение для тяжелых режимов
        Интервальный запуск нейросети (каждый 5-10 кадр)
        FPS до 60 в легких режимах
        Многопоточная обработка для детекции движения
    
    Автоматизация
        Motion Recording сам начинает/останавливает запись
        Файлы с временными метками
        Индикаторы движения на экране
        Графическая шкала уровня движения
        Пользовательский опыт
        Горячие клавиши для переключений
        Визуальные индикаторы (FPS, статус записи, эффекты)
        Выбор камеры при запуске
        Создание скриншотов одним нажатием
    
    Код
        Константы для настраиваемых параметров
        Комментарии на русском
        Обработка ошибок открытия камеры/файлов
        Кроссплатформенность

❌ Минусы

    Зависимости
        Body Skeleton требует ручного скачивания моделей (~200 МБ)
        DNN Face Detection требует файлов моделей
        Каскады Хаара не входят в репозиторий
    
    Производительность
        Однопоточная обработка в большинстве режимов
        Нет GPU ускорения для нейросетей
        Body Skeleton работает только на CPU
    
    Ограничения
        Color Tracking (базовый) видит только 1 объект
        Нет сохранения настроек между запусками
        Нет потокового видео-выхода
    
    Технические проблемы
        Устаревшие каскады Хаара (низкая точность в сложных условиях)
        Нет системы логирования ошибок
