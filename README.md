📷 Computer Vision Camera Application
Многофункциональное приложение компьютерного зрения для анализа видеопотока с веб-камеры в реальном времени.

🎮 Режимы работы

    Режим                            	Описание
    Simple Camera            	Базовый захват видео с FPS
    Motion Detection	            Детекция движения
    Color Tracking	        Отслеживание по цвету (G/R/B)
    Edge Detection	          Границы Canny (↑↓ пороги)
    Face Detection	                Лица + глаза
    Motion Recording        	Автозапись при движении (M/S)
    Body Skeleton	              Трекинг скелета (R/T)
    ESC -                      выход из любого режима

✅ Плюсы
Архитектура:
    
    Модульная структура с разделением на .h и .cpp
    Легко добавлять новые режимы
    Единый паттерн захват → обработка → вывод

Производительность:
    
    Уменьшенное разрешение для тяжелых режимов
    Интервальный запуск нейросети (каждый 5-10 кадр)
    FPS до 60 в легких режимах

Автоматизация:
    
    Motion Recording сам начинает/останавливает запись
    Файлы с временными метками
    Индикаторы движения на экране
    
 Пользовательский опыт:
    
    Горячие клавиши для переключений
    Визуальные индикаторы (FPS, статус записи)
    Графические шкалы движения

Код:
    
    Константы для настраиваемых параметров
    Комментарии на русском
    Обработка ошибок открытия камеры/файлов
    Кроссплатформенность

❌ Минусы
Зависимости:
    
    Body Skeleton требует ручного скачивания моделей (~200 МБ)
    Каскады Хаара не входят в репозиторий
    Жесткие пути к файлам (faceDetection сломается)

Производительность:
    
    Однопоточная обработка
    Нет GPU ускорения для нейросети

Ограничения:

    Color Tracking видит только 1 объект
    Нет выбора камеры (только индекс 0)
    Нет сохранения настроек между запусками
    Нет видео-выхода в файл с эффектами

Технические проблемы:        

    Нестандартный путь к OpenCV в headers.h
    Устаревшие каскады Хаара (низкая точность)
    Нет логирования ошибок

🔧 Что можно улучшить

    Добавить CUDA для ускорения нейросети
    Внедрить многопоточность с очередью кадров
    Заменить каскады Хаара на DNN детекторы (YOLO/SSD)
    Сделать выбор камеры через аргументы командной строки
    Добавить сохранение настроек в JSON/XML
    Реализовать запись обработанного видео
    Исправить жесткие пути к файлам моделей
    Убрать лишние копирования кадров

🛠️ Требования
    OpenCV 4.x+ (с модулем dnn)
    C++17
    4+ ГБ ОЗУ (8+ для скелета)
    Любая веб-камера

📁 Файлы моделей
    Для Face Detection:
      haarcascade_frontalface_default.xml
      haarcascade_eye.xml

Проект демонстрирует возможности OpenCV, но требует доработок для production использования
