📷 Computer Vision Camera Application
Многофункциональное приложение компьютерного зрения для анализа видеопотока с веб-камеры в реальном времени.

🎮 Режимы работы
Режим                            	Описание
Simple Camera            	Базовый захват видео с FPS
Motion Detection	            Детекция движения
Color Tracking	        Отслеживание по цвету (G/R/B)
Edge Detection	          Границы Canny (↑↓ пороги)
Face Detection	                Лица + глаза
Motion Recording        	Автозапись при движении (M/S)
Body Skeleton	              Трекинг скелета (R/T)
ESC -                      выход из любого режима

✅ Плюсы
Архитектура:
    1) Модульная структура с разделением на .h и .cpp
    2) Легко добавлять новые режимы
    3) Единый паттерн захват → обработка → вывод

Производительность:
    1) Уменьшенное разрешение для тяжелых режимов
    2) Интервальный запуск нейросети (каждый 5-10 кадр)
    3) FPS до 60 в легких режимах

Автоматизация:
    1) Motion Recording сам начинает/останавливает запись
    2) Файлы с временными метками
    3) Индикаторы движения на экране
    
 Пользовательский опыт:
     1) Горячие клавиши для переключений
     2) Визуальные индикаторы (FPS, статус записи)
     3)Графические шкалы движения

Код:
     1) Константы для настраиваемых параметров
     2) Комментарии на русском
     3) Обработка ошибок открытия камеры/файлов
     4) Кроссплатформенность

❌ Минусы
Зависимости:
    1) Body Skeleton требует ручного скачивания моделей (~200 МБ)
    2) Каскады Хаара не входят в репозиторий
    3) Жесткие пути к файлам (faceDetection сломается)

Производительность:
    1) Однопоточная обработка
    2) Нет GPU ускорения для нейросети

Ограничения:
    1) Color Tracking видит только 1 объект
    2) Нет выбора камеры (только индекс 0)
    3) Нет сохранения настроек между запусками
    4) Нет видео-выхода в файл с эффектами

Технические проблемы:
    1) Нестандартный путь к OpenCV в headers.h
    2) Устаревшие каскады Хаара (низкая точность)
    3) Нет логирования ошибок

🛠️ Требования
    OpenCV 4.x+ (с модулем dnn)
    C++17
    4+ ГБ ОЗУ (8+ для скелета)
    Любая веб-камера

    Для Face Detection:
      haarcascade_frontalface_default.xml
      haarcascade_eye.xml
